This is a prototype for NRC Playbook using Hugo.

Followed instructions here: https://gohugo.io/getting-started/quick-start/

## Setup

`brew install hugo`

## Run dev server

`hugo server -D`

## Build static site

`hugo -D`

Output will be in `public` directory by default.
